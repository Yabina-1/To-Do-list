
# ✅ To-Do List App (Python)

A simple, beginner-friendly command-line To-Do List app built with Python.

## 💡 Features
- Add new tasks
- Mark tasks as done
- Delete tasks
- Clear all tasks
- Track your progress

## 🚀 How to Run
1. Make sure Python is installed
2. Download `todo_app.py`
3. Open terminal and run:
   python todo_app.py

## 🛠️ Built With
- Python 3
- No external libraries needed!
